# Making Connections: Qt Object System and Object Communication

This is the repo for the **Making Connections: Qt Object System and Object Communication** Course on Qt Academy.

## Getting Started

To get started with this project, you will need to have Qt and an IDE (for example, Qt Creator) installed on your computer. You can download Qt and Qt Creator from the [Qt.io website](qt.io).

Once you have Qt and an IDE installed, head over to [Qt Academy](academy.qt.io) and follow along with the course.

Open the `CMakeLists.txt` file in Qt Creator, build and run the project. The application will launch a lumber mill control center where you can start, stop, and control the speed of a saw.
